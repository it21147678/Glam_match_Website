Introduction

We are dedicated to revolutionizing the cosmetic experience through cutting-edge mobile application solutions. Our mission is to simplify the complex process of selecting suitable cosmetic products by harnessing the power of artificial intelligence and advanced image processing technologies.

Our innovative platform automates the recommendation of lipstick colors, eyeliner styles, and foundation shades directly from user-uploaded photos, enabling individuals to optimize their beauty routines with greater efficiency and accuracy.

Research Problem & Solution

Research Problem

How can a mobile application, utilizing advanced image processing and machine learning techniques, effectively provide personalized recommendations for lipstick colors, foundation shades, and eyeliner styles based on a user's unique facial characteristics?

The widespread difficulty individuals experience in effectively selecting cosmetics that genuinely complement their unique features manifests in several key challenges:

  • Choosing correct lipstick colors that match skin tone and lip shape
  • Selecting foundation shades that adapt to varied lighting conditions
  • Determining eyeliner styles that align with individual eye shapes
  • Absence of precise, data-driven systems leading to user frustration

Proposed Solution

A revolutionary mobile application that offers personalized cosmetic suggestions through advanced image processing and machine learning techniques.

Users upload facial photos for preprocessing and quality enhancement

Advanced algorithms identify crucial facial features and skin characteristics

Machine learning models generate personalized cosmetic recommendations

Intuitive interface presents tailored suggestions with virtual try-on features

Research Objectives

1

Foundation Shade Recommendation

Recommend accurate and inclusive foundation shades that match individual skin tones.

Analyzes facial images to detect dominant skin tone using clustering techniques
Blends two shades when no exact match is found for personalized results
Accounts for different lighting conditions and environmental factors
Includes diverse dataset for global skin tone coverage
2

Personalized Lipstick Recommendations

Suggest personalized lipstick shades that complement the user's lip shape, skin tones, and facial features.

Identifies lip shapes (bow, full, heart-shaped) through advanced analysis
Detects skin tones using sophisticated color analysis algorithms
ML model maps features to suitable lipstick shades and finishes
Real-time virtual try-on with adaptive user feedback integration
3

Eyeliner Style Suggestions

Provide eyeliner style suggestions based on the user's specific eye shape and preferences.

Facial landmark detection and ML classification for eye shape analysis
Identifies eye shapes: almond, round, hooded, monolid variations
AR-based virtual try-on experience for real-time visualization
Integrates user preferences for continuous recommendation refinement

Methodology

The GlamMatch application employs a comprehensive methodology that integrates machine learning, advanced image processing, and user-centric design to deliver personalized cosmetic recommendations.

1

Image Acquisition

User-uploaded images undergo rigorous validation and normalization techniques to ensure consistent lighting and quality.

2

Feature Detection

Facial feature detection and segmentation using MediaPipe Face Mesh to extract relevant regions of interest.

3

Feature Extraction

Specific attributes like lip geometry, eye type, and skin tone are identified through clustering and classification algorithms.

4

ML Prediction

CNNs and Random Forest classifiers trained on diverse datasets predict suitable cosmetic options with blending algorithms.

5

Recommendation Engine

Cosmetic science rules applied for personalized suggestions with adaptive feedback system integration.

6

User Interface

Intuitive mobile interface with virtual try-on features, supported by robust Python Flask backend infrastructure.
